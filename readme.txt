Adds shortcodes to show Asana tasks in WordPress


== Description ==

This plugin allows you to show Asana tasks to users. Each user can view all tasks from a single workspace. Users can also add tasks to any project in their designated workspace.

WP Asana is particularly useful for developers and agencies who want to give clients limited access to their Asana workspace.

Please note that you will need an Asana account to use this plugin.

== Installation ==


1. Upload the `wp-asana` folder to your /wp-content/plugins/ directory
1. Activate the plugin through the Plugins menu
1. Go to the Asana User Settings page (Settings - Asana) to configure the plugin
